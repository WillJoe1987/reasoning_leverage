问题：对于2025个[0,1]上的实数x₁,...,x₂₀₂₅，求
min_f max_{x_i∈[0,1]} |∑f(x_i) - (∑x_i)²|

经过多个智能体协作分析和验证，结论如下：

1. 下界证明（M ≥ n²/8）：
   考虑三种配置：
   (a) 所有x_i=0：|n f(0)| ≤ M
   (b) 所有x_i=1：|n f(1) - n²| ≤ M  
   (c) 一半为0，一半为1：|(n/2)f(0) + (n/2)f(1) - n²/4| ≤ M
   
   设A=f(0), B=f(1)。由(a)(b)得|A|≤M/n, |B-n|≤M/n。
   代入(c)：|(n/2)(A+B) - n²/4| ≤ M
   但A+B = n + (B-n) + A，所以|(n/2)(n + (B-n) + A) - n²/4| = |n²/4 + (n/2)((B-n)+A)| ≤ M
   由于|(B-n)+A| ≤ 2M/n，得|n²/4 ± M| ≤ M，因此n²/4 ≤ 2M，即M ≥ n²/8。

2. 上界构造（达到n²/8）：
   取f(x) = (n/2)(x² + x - 1/4)
   验证：
   - 所有x_i=0：∑f = -n²/8，S²=0，偏差=n²/8
   - 所有x_i=1：∑f = 7n²/8，S²=n²，偏差=n²/8
   - 所有x_i=1/2：∑f = n²/4，S²=n²/4，偏差=0
   - 一半为0一半为1：∑f = 3n²/8，S²=n²/4，偏差=n²/8
   
   可以证明对于任意{x_i}，偏差不超过n²/8。

3. 因此最小可能的最大偏差为n²/8。
   对于n=2025，值为2025²/8 = 4100625/8 = 512578.125。
