**最优序列**：
\[

**验证**：
1. 严格递增：\(1 < 2 < \cdots < 36 < 665 < 666\) ✓
2. 总和：\(1+2+\cdots+36 + 665 + 666 = 666 + 1331 = 1997\) ✓

   - \(k = 38\)：\(b_{38} = 0\)
4. 因此 \(S = \max(b_k) = 666\)

### 下界证明（S ≥ 666）

**反证法**：假设存在序列使 \(S \leq 665\)。

记 \(L_k = \sum_{i=1}^k a_i\)，则 \(b_k = \min(L_k, 1997 - L_k)\)。

1. 由 \(b_{37} \leq 665\) 且 \(L_{37} \geq 1+2+\cdots+37 = 703\)，得 \(a_{38} \leq 665\)。
   （否则 \(b_{37} = \min(L_{37}, a_{38}) \geq \min(703, 666) = 666 > 665\)）

2. 于是 \(L_{37} = 1997 - a_{38} \geq 1997 - 665 = 1332\)。

3. 考虑 \(b_{36} \leq 665\)：
   \[
   b_{36} = \min(L_{36}, a_{37}+a_{38})
   \]
   若 \(L_{36} \leq 665\)，则 \(L_{36} = 1997 - (a_{37}+a_{38}) \leq 665\) 
   ⇒ \(a_{37}+a_{38} \geq 1332\)。
   但 \(a_{37} \leq a_{38}-1 \leq 664\)，故 \(a_{37}+a_{38} \leq 664+665 = 1329 < 1332\)，矛盾。
   因此必须 \(a_{37}+a_{38} \leq 665\)。

4. 由 \(a_{37}+a_{38} \leq 665\) 得 \(a_{37} \leq 665 - a_{38}\)。

5. 另一方面：
   \[
   L_{37} = \sum_{i=1}^{36} a_i + a_{37} \geq (1+2+\cdots+36) + a_{37} = 666 + a_{37}
   \]
   结合 \(L_{37} \geq 1332\)，得 \(666 + a_{37} \geq 1332\) ⇒ \(a_{37} \geq 666\)。

6. 步骤4给出 \(a_{37} \leq 665 - a_{38} \leq 665\)，与步骤5的 \(a_{37} \geq 666\) 矛盾。

因此假设 \(S \leq 665\) 不成立，故 \(S \geq 666\)。
