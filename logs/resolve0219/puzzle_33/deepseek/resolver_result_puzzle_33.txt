# 相册问题的最小人数证明（严谨版）

## 问题重述
相册有25张照片，每张显示三人(Lᵢ, Mᵢ, Rᵢ)，满足：
1. Lᵢ是Mᵢ的亲兄弟，Rᵢ是Mᵢ的女儿
2. M₁,...,M₂₅各不相同
3. 相册中任意两人没有共同的孩子

求最少不同人数。

## 严谨证明

### 符号
- M = {M₁,...,M₂₅}：中间人集合
- R = {R₁,...,R₂₅}：右边人（女儿）集合
- L = {L₁,...,L₂₅}：左边人（兄弟）集合

### 引理1：|M| = 25
由条件(2)，M各不相同，故|M|=25。

### 引理2：|R| = 25 且 R中元素各不相同
对任意i≠j，若Rᵢ = Rⱼ，则Mᵢ和Mⱼ有共同女儿Rᵢ=Rⱼ，违反条件(3)。故所有Rᵢ互异，|R|=25。

### 引理3：M ∩ R = ∅
∀i,j：Mᵢ是男性（因他是某人兄弟），Rⱼ是女性（因她是某人女儿），故Mᵢ ≠ Rⱼ。

### 定理1：总人数 ≥ 50
由引理1-3：|M∪R| = |M| + |R| = 25+25=50。
因M∪R ⊆ M∪L∪R，故|M∪L∪R| ≥ 50。

### 定理2：存在构造使总人数=50
构造：
1. 25个男性B₁,...,B₂₅，互为亲兄弟（同父母）
2. 25个女性D₁,...,D₂₅
3. 关系：Bᵢ是Dᵢ的父亲（i=1,...,25）
4. 照片安排：
   - i=1,...,24：(Lᵢ,Mᵢ,Rᵢ) = (Bᵢ₊₁, Bᵢ, Dᵢ)
   - i=25：(L₂₅,M₂₅,R₂₅) = (B₁, B₂₅, D₂₅)

验证：
- 条件(1)：∀i，Lᵢ是Mᵢ亲兄弟（所有B互为兄弟），Rᵢ是Mᵢ女儿（Bᵢ是Dᵢ父亲）✓
- 条件(2)：M₁,...,M₂₅ = B₁,...,B₂₅互异 ✓
- 条件(3)：任意两人无共同孩子：
  * ∀i≠j，Bᵢ和Bⱼ：女儿分别为Dᵢ和Dⱼ，不同，故无共同孩子
  * Bᵢ和Dⱼ：若i≠j，无亲子关系；若i=j，Bᵢ是Dᵢ父亲，但Dᵢ母亲不在相册，故Bᵢ和Dᵢ无共同孩子
  * Dᵢ和Dⱼ：无亲子关系 ✓

### 结论
由定理1（下界50）和定理2（可达50），得最小人数为50。

## 关键点澄清
1. "女儿"⇒女性，"兄弟"⇒男性，故M/R性别不同，集合不相交
2. "无共同孩子"⇒若两人都是某孩子父母，则禁止；仅一人是父母不违反
3. 兄弟关系可在男性内部满足，不增新人
4. 构造中所有女儿不同，满足条件(3)

## 最终答案：50
