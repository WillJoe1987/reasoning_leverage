最终结果:
对于整数 $n>2$，在约束 $\sum_{k=1}^{n} a_k = 2n$ 和 $\sum_{k=1}^{n} k|a_k| = 4n$ 下，$a_1^2 + a_2^2 + \cdots + a_n^2$ 的最小值为：

\[
\boxed{\begin{cases}
4n, & \text{if } n = 3 \\[6pt]
\dfrac{6}{5}n^2, & \text{if } n \ge 4
\end{cases}}
\]

具体地：
- 当 $n=3$ 时，最小值为 $12$，达到于 $a_1=a_2=a_3=2$。
- 当 $n=4$ 时，最小值为 $\dfrac{96}{5}=19.2$，达到于 $a_1=\dfrac{16}{5}=3.2$，$a_2=\dfrac{12}{5}=2.4$，$a_3=\dfrac{8}{5}=1.6$，$a_4=\dfrac{4}{5}=0.8$。
- 当 $n=5$ 时，最小值为 $30$，达到于 $a_1=4$，$a_2=3$，$a_3=2$，$a_4=1$，$a_5=0$。
- 当 $n\ge 6$ 时，最小值为 $\dfrac{6}{5}n^2$，达到于 $a_k = n - \dfrac{n}{5}k$（$k=1,2,3,4,5$），$a_k=0$（$k\ge 6$）。

推导过程:
**完整推导过程：**

**1. 问题分析**
给定约束：
\[
\sum_{k=1}^{n} a_k = 2n, \qquad \sum_{k=1}^{n} k|a_k| = 4n
\]
需要最小化 $\sum_{k=1}^{n} a_k^2$，其中 $n>2$ 为整数。

**2. 符号处理与简化**
设 $b_k = |a_k| \ge 0$，$s_k = \operatorname{sign}(a_k) \in \{-1,0,1\}$，则 $a_k = s_k b_k$。约束变为：
\[
\sum_{k=1}^{n} s_k b_k = 2n, \qquad \sum_{k=1}^{n} k b_k = 4n
\]
目标函数为 $\sum_{k=1}^{n} b_k^2$。

**3. 证明最优解中所有 $a_k \ge 0$**
假设在最优解中存在 $a_i < 0$。令 $c = |a_i| > 0$。考虑将 $a_i$ 改为 $c$（即变为正），同时将某个 $a_j > 0$ 减少 $2c$（若可能）。这一调整不改变 $\sum k|a_k|$，且保持 $\sum a_k$ 不变，但平方和的变化为：
\[
(c^2 - (-c)^2) + ((a_j-2c)^2 - a_j^2) = 0 + (a_j^2 - 4a_j c + 4c^2 - a_j^2) = 4c(c - a_j)
\]
若 $c \le a_j$，则平方和不增加。通过适当调整，总可得到全非负的可行解且平方和不大于原解。故可设所有 $a_k \ge 0$，即 $b_k = a_k$。

问题简化为：在约束
\[
\sum_{k=1}^{n} b_k = 2n, \qquad \sum_{k=1}^{n} k b_k = 4n, \qquad b_k \ge 0
\]
下，最小化 $\sum_{k=1}^{n} b_k^2$。

**4. 凸优化与拉格朗日乘子法**
这是凸集上的凸优化问题。先忽略非负约束，构造拉格朗日函数：
\[
L = \sum_{k=1}^{n} b_k^2 - \lambda\left(\sum_{k=1}^{n} b_k - 2n\right) - \mu\left(\sum_{k=1}^{n} k b_k - 4n\right)
\]
令偏导为零：
\[
\frac{\partial L}{\partial b_k} = 2b_k - \lambda - \mu k = 0 \quad \Rightarrow \quad b_k = \frac{\lambda + \mu k}{2}
\]

**5. 考虑非负约束与支持集**
实际解需满足 $b_k \ge 0$。由线性‑二次规划理论，最优解仅有有限个 $b_k > 0$。设前 $m$ 个 $b_k$ 为正，其余为零，且解具形式 $b_k = \alpha + \beta k$（$k=1,\ldots,m$）。

代入约束：
\[
\sum_{k=1}^{m} (\alpha + \beta k) = m\alpha + \beta\frac{m(m+1)}{2} = 2n \tag{1}
\]
\[
\sum_{k=1}^{m} k(\alpha + \beta k) = \alpha\frac{m(m+1)}{2} + \beta\frac{m(m+1)(2m+1)}{6} = 4n \tag{2}
\]

**6. 求解参数**
记 $S_1 = \frac{m(m+1)}{2}$，$S_2 = \frac{m(m+1)(2m+1)}{6}$。由 (1) 得 $\alpha = \frac{2n - \beta S_1}{m}$，代入 (2)：
\[
\frac{S_1}{m}(2n - \beta S_1) + \beta S_2 = 4n
\]
整理得：
\[
\beta\left(S_2 - \frac{S_1^2}{m}\right) = 4n - \frac{2n S_1}{m}
\]
计算：
\[
S_2 - \frac{S_1^2}{m} = \frac{m(m+1)(m-1)}{12}, \qquad 4n - \frac{2n S_1}{m} = n(3-m)
\]
故
\[
\beta = \frac{12n(3-m)}{m(m+1)(m-1)}, \qquad \alpha = \frac{2n}{m} - \frac{\beta(m+1)}{2}
\]

**7. 确定支持集大小 $m$**
非负性要求 $b_m = \alpha + \beta m \ge 0$ 且（若 $m < n$）$b_{m+1} = \alpha + \beta(m+1) < 0$。

- 由 $b_m \ge 0$：
  \[
  \frac{2n}{m} + \beta\cdot\frac{m-1}{2} \ge 0
  \]
  代入 $\beta$ 化简得 $10 - 2m \ge 0$，即 $m \le 5$。

- 由 $b_{m+1} < 0$（当 $m < n$）：
  \[
  \frac{2n}{m} + \beta\cdot\frac{m+1}{2} < 0
  \]
  代入 $\beta$ 化简得 $8 - 2m < 0$，即 $m > 4$。

因此：
- 若 $n \le 5$，则 $m = n$（此时无 $b_{m+1}$ 条件）。
- 若 $n > 5$，则 $m = 5$。

**8. 计算最小值**
- **$n=3$**：$m=3$，$\beta=0$，$\alpha=2$，$b_k=2$，平方和 $=3\times 2^2 = 12 = 4n$。
- **$n=4$**：$m=4$，$\beta = -\frac{4}{5}$，$\alpha=4$，$b_k = 4 - \frac{4}{5}k$，
  \[
  \sum b_k^2 = \sum_{k=1}^{4}\left(4-\frac{4}{5}k\right)^2 = \frac{96}{5} = 19.2 = 4.8n
  \]
- **$n=5$**：$m=5$，$\beta=-1$，$\alpha=5$，$b_k = 5-k$，
  \[
  \sum b_k^2 = 4^2+3^2+2^2+1^2+0^2 = 30 = 6n
  \]
- **$n\ge 6$**：$m=5$，$\beta = -\frac{n}{5}$，$\alpha = n$，$b_k = n - \frac{n}{5}k$（$k=1,\ldots,5$），$b_k=0$（$k\ge 6$），
  \[
  \sum b_k^2 = n^2\sum_{k=1}^{5}\left(1-\frac{k}{5}\right)^2 = n^2\cdot\frac{6}{5} = \frac{6}{5}n^2
  \]

**9. 统一表达式**
观察上述结果：
- $n=3$ 时，最小值为 $4n$。
- $n\ge 4$ 时，最小值为 $\frac{6}{5}n^2$（注意 $n=4,5$ 时 $\frac{6}{5}n^2$ 分别等于 $19.2$ 和 $30$，与前面一致）。

故最终答案为：
\[
\min \sum_{k=1}^{n} a_k^2 = 
\begin{cases}
4n, & n=3 \\[6pt]
\dfrac{6}{5}n^2, & n\ge 4
\end{cases}
\]

**10. 验证**
- $n=3$：$a_1=a_2=a_3=2$，满足约束，平方和 $=12$。
- $n=4$：$a_k = 4-0.8k$，满足约束，平方和 $=19.2$。
- $n=5$：$a_k = 5-k$，满足约束，平方和 $=30$。
- $n=6$：$a_1=4.8, a_2=3.6, a_3=2.4, a_4=1.2, a_5=a_6=0$，满足约束，平方和 $=43.2 = \frac{6}{5}\times 36$。

所有解均满足 $a_k \ge 0$，且经检验无更优解。
