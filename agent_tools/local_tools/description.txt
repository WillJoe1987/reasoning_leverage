**大小**: 6627字符
**状态**: 未被git忽略

#### 功能描述
- **主要功能**: 判断指定文件或目录是否被git忽略
- **工具类型**: LangChain工具装饰器（@tool）
- **检查方法**: 双重检查策略
  1. 优先使用`git check-ignore -v`命令
  2. 回退到手动解析.gitignore文件

#### 接口说明
- **函数**: `is_git_ignored(path: str) -> dict`
- **参数**: `path` - 要检查的文件或目录路径
- **返回值**: 包含详细信息的字典结构：
  ```python
  {
      "path": str,           # 输入的路径
      "git_root": str,       # Git根目录（如果找到）
      "ignored": bool,       # 是否被忽略
      "matched_pattern": str, # 匹配的.gitignore模式
      "method": str,         # 使用的检查方法（"git"或"parse"）
      "git_error": str,      # 可选的git错误信息
      "error": str           # 可选的异常错误信息
  }
  ```

### 2. caculator.py - 安全数学计算工具集
**路径**: `caculator.py`
**大小**: 10005字符
**状态**: 未被git忽略

#### 功能概述
该文件是一个**安全数学计算工具集**，为智能体系统提供安全的数学计算能力，包含三个核心功能：
1. **安全数学表达式计算** - 支持单行数学表达式的安全计算
2. **迭代任务求解** - 支持在指定范围内迭代计算表达式
3. **纯数学代码沙盒** - 提供安全的Python代码执行环境，专门用于数学计算和算法验证

#### 类结构分析

##### SafeMathEvaluator类 - 安全数学表达式计算器
- **功能**: 安全地计算数学表达式，防止恶意代码执行
- **安全特性**:
  - 白名单函数机制：只允许预定义的数学函数
  - AST语法树检查：递归检查所有节点
  - 支持的运算符：算术、比较、逻辑运算符
  - 支持的函数：基础数学、三角函数、对数函数等

##### IterativeTaskSolver类 - 迭代任务求解器
- **功能**: 使用SafeMathEvaluator作为底层引擎，安全地执行声明式迭代任务
- **任务描述格式**: JSON格式，包含expression、variable、range、condition、aggregate等字段

##### PureMathSandbox类 - 纯数学代码沙盒
- **功能**: 提供严格受限的Python代码执行环境
- **安全限制**:
  - 禁止导入、文件、网络、系统操作
  - 禁止类定义、异常处理、异步操作
  - 只允许math、random模块和基础内置函数
  - 支持print输出捕获（限制2048字符）

#### 工具接口

##### 计算器工具集 - get_calculator_tool()
**返回工具**:
1. `tool_calculator(expression_line: str) -> Any`
   - 功能: 安全计算单行数学表达式
   - 参数: `expression_line` - 数学表达式字符串

2. `tool_iterative_solve(task_description: str) -> str`
   - 功能: 执行迭代计算任务
   - 参数: `task_description` - JSON格式的任务描述字符串

##### 数学沙盒工具 - get_sandbox_tool()
**返回工具**:
- `tool_math_sandbox(code: str, inputs_json: str = '{}') -> str`
  - 功能: 在安全沙盒中执行数学计算代码
  - 参数: `code` - Python代码片段，`inputs_json` - JSON格式的输入变量字典

#### 业务角色
- **数学计算引擎**: 为智能体提供安全的数学计算能力
- **算法验证**: 支持复杂的数学验证和算法测试
- **安全保障**: 通过多层安全检查，防止恶意代码执行
