## Problem Solution: 19×19 Board Stone Flipping Problem

### Problem Statement
On a 19×19 board with 361 squares, each containing a black or white stone. An operation consists of choosing one row and one column and reversing the color of all 37 stones (19 in row + 19 in column - 1 for the intersection counted once). Find the largest n such that we can always achieve at least n black stones.

### Mathematical Formulation
- Represent the board as a 19×19 matrix A over GF(2), where 0=white, 1=black
- An operation on row i and column j adds matrix M where M_rc = 1 if r=i or c=j
- The operation space consists of matrices of form r_i + c_j (mod 2) for vectors r, c ∈ {0,1}^19
- From initial board A, reachable boards are B = A + M where M is in the operation space

### Lower Bound Proof
For any initial configuration A:
1. Choose r = first column of A (a₁)
2. For each column j, we can choose c_j to maximize black stones
3. Column 1: wt(a₁ + r) = wt(a₁ + a₁) = 0, so we get max(0, 19) = 19 black stones
4. Columns 2-19: wt(a_j + r) = wt(a_j + a₁), contributing max(k, 19-k) ≥ 10 black stones each
5. Total: 19 + 18 × 10 = **199** black stones

### Tightness Proof
To show 199 is the best possible lower bound, we need a configuration where maximum = 199:
- Requires 19 vectors in {0,1}^19 with pairwise Hamming distances 9 or 10
- For m=19 (odd), such codes exist via quadratic residue construction
- Using quadratic residues mod 19: {1, 4, 5, 6, 7, 9, 11, 16, 17}
- Construct 19 vectors where vector i has 1 at position j iff (j-i) mod 19 is a quadratic residue
- All pairwise distances equal 10 ∈ {9, 10}
- For such A, the maximum achievable is exactly 199

### Verification
- Small cases verified: m=1→1, m=3→7 (both match formula m + (m-1)×ceil(m/2))
- Two independent verification agents confirmed the answer 199
- Formula for odd m: m + (m-1)×(m+1)/2 = 19 + 18×10 = 199

### Final Answer
The largest integer n such that we can always achieve at least n black stones is **199**.
