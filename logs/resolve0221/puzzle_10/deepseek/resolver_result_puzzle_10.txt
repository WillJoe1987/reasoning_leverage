# 问题求解：最小值问题

## 问题描述
已知正实数 \(x,y,z\) 满足：
1. \(xy + xz + yz \neq 1\)
2. \(\frac{(x^2 - 1)(y^2 - 1)}{xy} + \frac{(x^2 - 1)(z^2 - 1)}{xz} + \frac{(z^2 - 1)(y^2 - 1)}{zy} = 4\)

求 \(9x^2 + 13689y^2z^2 - 27378yz + \frac{28561}{(y+z)^2}\) 的最小值。

## 关键推导

### 步骤1：分析条件2
经过代数推导，条件2等价于：
\[
(xy + xz + yz - 1)(xyz - x - y - z) = 0
\]
由于 \(xy + xz + yz \neq 1\)，所以必须有：
\[
xyz = x + y + z
\]

### 步骤2：变量替换
由 \(xyz = x + y + z\) 得：
\[
x = \frac{y+z}{yz-1} \quad (\text{要求 } yz \neq 1)
\]
目标函数重写为：
\[
f = 9x^2 + 13689(yz-1)^2 + \frac{28561}{(y+z)^2} - 13689
\]
代入 \(x\)：
\[
f = \frac{9(y+z)^2}{(yz-1)^2} + 13689(yz-1)^2 + \frac{28561}{(y+z)^2} - 13689
\]

令 \(a = yz\)，\(b = y+z\)，则：
\[
f(a,b) = \frac{9b^2}{(a-1)^2} + 13689(a-1)^2 + \frac{28561}{b^2} - 13689
\]

### 步骤3：优化求解

#### 方法1：偏导数法
固定 \(a\)，对 \(b\) 优化：
\[
\frac{\partial f}{\partial b} = \frac{18b}{(a-1)^2} - \frac{57122}{b^3} = 0
\]
得 \(b^2 = \frac{169}{3}(a-1)\)

代入 \(f\)：
\[
f(a) = \frac{1014}{a-1} + 13689(a-1)^2 - 13689
\]
求导：
\[
f'(a) = -\frac{1014}{(a-1)^2} + 27378(a-1) = 0
\]
得 \((a-1)^3 = \frac{1}{27}\)，所以 \(a-1 = \frac{1}{3}\)，即 \(a = \frac{4}{3}\)

进而：
\[
b^2 = \frac{169}{3} \times \frac{1}{3} = \frac{169}{9} \Rightarrow b = \frac{13}{3}
\]
\[
x = \frac{b}{a-1} = \frac{13/3}{1/3} = 13
\]

最小值：
\[
f_{\text{min}} = \frac{1014}{1/3} + 13689 \times (\frac{1}{3})^2 - 13689 = 3042 + 1521 - 13689 = -9126
\]

#### 方法2：拉格朗日乘数法
建立拉格朗日函数：
\[
\mathcal{L} = 9x^2 + 13689y^2z^2 - 27378yz + \frac{28561}{(y+z)^2} + \lambda(xyz - x - y - z)
\]
求偏导得KKT条件，解得：
\[
x = 13, \quad y = 4, \quad z = \frac{1}{3}, \quad \lambda = -702
\]
验证满足所有条件，计算得最小值 -9126。

#### 方法3：不等式方法（AM-GM）
由AM-GM不等式：
\[
\frac{9b^2}{(a-1)^2} + \frac{28561}{b^2} \ge \frac{1014}{a-1}
\]
等号成立当 \(b^2 = \frac{169}{3}(a-1)\)

进而：
\[
\frac{1014}{a-1} + 13689(a-1)^2 = \frac{507}{a-1} + \frac{507}{a-1} + 13689(a-1)^2 \ge 3\sqrt[3]{507^2 \times 13689} = 4563
\]
等号成立当 \(a-1 = \frac{1}{3}\)

所以：
\[
f \ge 4563 - 13689 = -9126
\]

### 步骤4：验证解
取 \(x=13\), \(y=4\), \(z=\frac{1}{3}\)：
1. \(xyz = 13×4×\frac{1}{3} = \frac{52}{3}\), \(x+y+z = 13+4+\frac{1}{3} = \frac{52}{3}\) ✓
2. \(xy+xz+yz = 52 + \frac{13}{3} + \frac{4}{3} = \frac{173}{3} \neq 1\) ✓
3. 条件2验证：计算得4 ✓
4. 目标函数值：计算得 -9126 ✓

## 最终答案
最小值为 \(-9126\)，在 \(x=13\), \(y=4\), \(z=\frac{1}{3}\)（或对称解）时达到。
