# 完整推导过程

## 问题重述
初始黑板上有一个正整数n（1≤n≤10000）。玩家轮流操作：擦掉一个大于1的整数k，用5个正整数替换，这5个数的和等于k，且任意两数的差不超过1。无法操作时当前玩家输。双方最优策略，玩家1先手且获胜。求可能的n值数量。

## 组合博弈论分析

### 1. Grundy值定义
定义g(k)为数字k的Grundy值。对于只有一个数字的游戏，整个游戏的Grundy值就是g(k)。先手必胜当且仅当g(k)≠0。

### 2. 基本位置
- k=1：无法操作，g(1)=0
- k=2,3,4：没有合法替换（需要5个正整数，最小和是5），g(2)=g(3)=g(4)=0

### 3. 操作分析
对于k≥5，设k=5a+t，其中a=⌊k/5⌋≥1，0≤t≤4。

合法替换要求5个正整数，任意两数差≤1。设最小值为x，则5个数是x或x+1。
设其中有m个x+1，则5-m个x。总和：k=5x+m，所以m=k-5x。

由于k=5a+t，有k-5x=t-5(x-a)，所以x必须满足：x=a或x=a-1（当t>0时）。

具体替换方案：
- t=0：k=5a，唯一替换：全部为a
- t=1：k=5a+1，替换：4个a，1个a+1
- t=2：k=5a+2，替换：3个a，2个a+1
- t=3：k=5a+3，替换：2个a，3个a+1
- t=4：k=5a+4，替换：1个a，4个a+1

### 4. Grundy值递归公式推导
计算每个替换的nim和（⊕表示异或）：
- t=0：nim和 = g(a)⊕g(a)⊕g(a)⊕g(a)⊕g(a) = g(a)
- t=1：nim和 = g(a)⊕g(a)⊕g(a)⊕g(a)⊕g(a+1) = g(a+1)
- t=2：nim和 = g(a)⊕g(a)⊕g(a)⊕g(a+1)⊕g(a+1) = g(a)
- t=3：nim和 = g(a)⊕g(a)⊕g(a+1)⊕g(a+1)⊕g(a+1) = g(a+1)
- t=4：nim和 = g(a)⊕g(a+1)⊕g(a+1)⊕g(a+1)⊕g(a+1) = g(a)

所以g(k)=mex(S)，其中S是单元素集合：
- t∈{0,2,4}：S={g(a)}
- t∈{1,3}：S={g(a+1)}

由于mex{0}=1，mex{1}=0，得到递归公式：
对于k≥5：
- 如果t∈{0,2,4}：g(k)=1 if g(a)=0 else 0
- 如果t∈{1,3}：g(k)=1 if g(a+1)=0 else 0

### 5. 动态规划计算
使用动态规划计算k=1到10000的g(k)：

初始化：
g[1]=g[2]=g[3]=g[4]=0

对于k从5到10000：
a = k // 5
t = k % 5
if t in {0,2,4}:
    g[k] = 1 if g[a]==0 else 0
else: # t in {1,3}
    g[k] = 1 if g[a+1]==0 else 0

### 6. 计算结果
统计g(k)=1（先手必胜）的k的数量：
- 总范围：1到10000
- 必胜位置数量：7656
- 必败位置数量：2344

### 7. 模式验证与关键观察
1. g(k)∈{0,1} 对所有k
2. 最后一个必败位置：k=3124
3. 对于k≥3125，所有g(k)=1
4. 最大连续必败区间：625到2500，共1876个数
5. 关键值验证：
   - g(1)=0, g(5)=1

### 8. 最终结论
玩家1有必胜策略的n值数量为7656。
