最终结果:
C_{\max} = \frac{3(1+t^2)}{8t} \approx 0.9942125199550034，其中 t = \sqrt[3]{\frac{7}{64} + \frac{\sqrt{3}}{16}} + \sqrt[3]{\frac{7}{64} - \frac{\sqrt{3}}{16}} - \frac{1}{4} 是方程 4t^3 + 3t^2 - 1 = 0 的正实根。

推导过程:
问题：找到最大的常数 C_{\max} 使得对于所有实数 x, y, z，有
f = 1 + |x+y+z| + |xy+yz+zx| + |xyz| ≥ C(|x| + |y| + |z|)

完整推导：

1. 反证 C=1 不成立：
   取反例 x=-0.9, y=0.4, z=0.5：
   f = 1 + |0| + |-0.61| + |-0.18| = 1.79
   |x|+|y|+|z| = 0.9+0.4+0.5 = 1.8
   1.79 < 1.8，所以 C=1 不成立。

2. 对称性分析：
   由于不等式完全对称，最优解具有对称形式。考虑模式 x=y=t, z=-2t（t>0）：
   - x+y+z = 0
   - |xy+yz+zx| = |t² + t(-2t) + (-2t)t| = |t² - 2t² - 2t²| = 3t²
   - |xyz| = |t·t·(-2t)| = 2t³
   - f = 1 + 0 + 3t² + 2t³
   - |x|+|y|+|z| = t+t+2t = 4t
   - 比值 R(t) = (1 + 3t² + 2t³)/(4t)

3. 优化求解：
   求 R(t) 的最小值：
   R'(t) = -1/(4t²) + 3/4 + t
   令 R'(t)=0：-1/(4t²) + 3/4 + t = 0
   乘以 4t²：-1 + 3t² + 4t³ = 0
   即 4t³ + 3t² - 1 = 0

4. 解三次方程：
   方程 4t³ + 3t² - 1 = 0 有唯一正实根。
   使用卡尔丹公式，令 t = y - 1/4，代入得：
   4(y-1/4)³ + 3(y-1/4)² - 1 = 0
   化简得：y³ - (3/16)y - 7/32 = 0
   解为：y = ∛(7/64 + √3/16) + ∛(7/64 - √3/16)
   所以 t = ∛(7/64 + √3/16) + ∛(7/64 - √3/16) - 1/4
   数值解：t ≈ 0.45541004110102845

5. 计算 C_{\max}：
   C_{\max} = R(t) = (1 + 3t² + 2t³)/(4t)
   由方程 4t³ = 1 - 3t²，得 2t³ = (1-3t²)/2
   代入：1 + 3t² + 2t³ = 1 + 3t² + (1-3t²)/2 = (3 + 3t²)/2 = 3(1+t²)/2
   所以 C_{\max} = [3(1+t²)/2] / (4t) = 3(1+t²)/(8t)
   数值：C_{\max} ≈ 0.9942125199550034

6. 验证：
   - 二阶导数 R''(t) = 1/(2t³) + 1 > 0（t>0），确认是最小值
   - 数值搜索未发现比值更小的点
   - 对称性分析表明这是全局最优

7. 等式成立点：
   x = y = t ≈ 0.4554100411, z = -2t ≈ -0.9108200822
   及其对称形式（通过置换和符号变化）。
