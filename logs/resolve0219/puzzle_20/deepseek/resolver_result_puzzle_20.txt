推导过程:
问题：学校有3个新班级。对于任意两个来自不同班级的学生，在第三个班级中恰好有10个学生同时认识他们，且恰好有10个学生既不认识他们中的任何一个。求总人数。

解：

设三个班级A、B、C的学生数分别为a、b、c。

对于任意x∈A, y∈B，设：
- N(x) = C班中认识x的学生集合
- N(y) = C班中认识y的学生集合

已知：
1. |N(x) ∩ N(y)| = 10
2. |N(x)ᶜ ∩ N(y)ᶜ| = 10

由包含排斥原理：
|N(x) ∪ N(y)| = |N(x)| + |N(y)| - |N(x) ∩ N(y)| = d_A + d_B - 10
其中d_A = |N(x)|，d_B = |N(y)|

同时：
|N(x) ∪ N(y)| = c - |N(x)ᶜ ∩ N(y)ᶜ| = c - 10

因此：
d_A + d_B - 10 = c - 10
⇒ d_A + d_B = c

由于这对任意x∈A, y∈B成立，所以d_A和d_B都是常数。

由对称性，类似可得：
d_A + d_C = b
d_B + d_C = a
其中d_C是每个C班学生在A班（或B班）中认识的人数。

边数守恒：
a × d_A = c × d_C
a × d_A = b × d_B
b × d_B = c × d_C

设这个公共值为K。

对称性分析：
由问题的完全对称性，假设a = b = c = n是最自然的。

那么：
d_A + d_B = n
d_A + d_C = n
d_B + d_C = n

解得：d_A = d_B = d_C = n/2

利用共同邻居条件：
对于任意x∈A, y∈B，同时认识x和y的C班学生数 = 10。

计算所有对(x,y)的λ(x,y)之和：
一方面，对于固定x∈A，∑_{y∈B} λ(x,y) = C班中认识x的学生与所有B班学生的认识关系数 = (n/2) × (n/2) = n²/4

对所有x∈A求和：n × (n²/4) = n³/4

另一方面，由于每个λ(x,y) = 10，总和 = n × n × 10 = 10n²

因此：
n³/4 = 10n²
⇒ n/4 = 10（因为n > 0）
⇒ n = 40

验证：
每个班级40人，每个学生认识其他班级中的20人。
对于任意x∈A, y∈B：
- C班中认识x的学生数：20
- C班中认识y的学生数：20
- 同时认识x和y的学生数：10
- 认识x或y的学生数：20 + 20 - 10 = 30
- 既不认识x也不认识y的学生数：40 - 30 = 10

符合条件。

总人数：3 × 40 = 120

注：存在更一般的解满足(d_AC - 10)(d_BC - 10) = 100，但由问题的完全对称性，对称解n=40是最自然且最可能的解。
